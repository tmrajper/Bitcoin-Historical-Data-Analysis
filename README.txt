To download all the resources to run the project, follow this link and clone the repository.

https://github.com/tmrajper/Bitcoin-Historical-Data-Analysis

Once downloaded, the dataset has already been purged of all missing data, so run the program like any other python/anacondas program.

Besure to have the program and the dataset in the same directory, or program will not find the dataset and crash.
Due to the sheer size of the data, Line graphs seemed to be the best representation. 

There is also a link to the dataset in the powerpoint presentation.
